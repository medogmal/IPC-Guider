# PPE Visual Guide Images

## Required Images

### donning_doffing_guide.png
- **Purpose**: Visual guide showing the proper sequence for donning (putting on) and doffing (removing) personal protective equipment
- **Recommended size**: 1200x800 pixels or similar aspect ratio
- **Format**: PNG or JPG
- **Content should include**:
  - Step-by-step visual demonstration
  - Clear illustrations of each PPE item
  - Proper sequence indicators
  - Do's and Don'ts examples

## How to Add Your Image

1. Create or obtain a PPE donning/doffing guide image
2. Name it `donning_doffing_guide.png`
3. Place it in this `assets/images/` directory
4. The app will automatically display it in the General Principles screen

## Image Guidelines

- Use clear, high-contrast images
- Include visual indicators for sequence (numbers, arrows)
- Show both donning and doffing procedures
- Consider healthcare professional audience
- Ensure medical accuracy of all PPE demonstrations
