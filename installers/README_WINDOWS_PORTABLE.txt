ABOUT:
------
IPC Guider is an offline-first infection prevention and control application
designed for healthcare professionals. It provides evidence-based calculators,
protocols, outbreak management tools, and comprehensive IPC guidelines.

SYSTEM REQUIREMENTS:
--------------------
- Operating System: Windows 10 (64-bit) or Windows 11
- RAM: 4 GB minimum, 8 GB recommended
- Disk Space: 100 MB free space
- Display: 1280x720 minimum resolution
- Internet: NOT required (fully offline)

INSTALLATION:
-------------
This is a PORTABLE version - no installation required!

1. Extract the ZIP file to any location on your computer
   (e.g., C:\Program Files\IPC Guider or Desktop\IPC Guider)

2. Double-click "ipc_guider.exe" to run the application

3. (Optional) Create a desktop shortcut:
   - Right-click on "ipc_guider.exe"
   - Select "Send to" > "Desktop (create shortcut)"

IMPORTANT FILES:
----------------
- ipc_guider.exe              : Main application executable
- flutter_windows.dll         : Flutter framework (required)
- *.dll files                 : Plugin libraries (required)
- data folder                 : App assets and resources (required)

DO NOT DELETE any files or folders - all are required for the app to work!

USAGE:
------
1. Launch the app by double-clicking "ipc_guider.exe"
2. Navigate through modules using the bottom navigation bar
3. Use calculators to compute infection rates and metrics
4. Save results for future reference
5. Export data to Excel or CSV for reporting

TROUBLESHOOTING:
----------------
Problem: App doesn't start
Solution: Make sure all DLL files are in the same folder as ipc_guider.exe

Problem: Missing features or data
Solution: Ensure the "data" folder is present and not empty

Problem: Windows SmartScreen warning
Solution: Click "More info" then "Run anyway" - the app is safe

Problem: Antivirus blocking the app
Solution: Add ipc_guider.exe to your antivirus whitelist/exceptions

UNINSTALLATION:
---------------
Simply delete the entire folder - no registry entries or system files are
created. Your saved data is stored in:
%APPDATA%\com.dryazeed\ipc_guider\
